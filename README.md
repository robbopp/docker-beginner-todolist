# Docker Beginner Todo List

A simple Todo List application containerized with Docker.  
This project is aimed at beginners learning Docker and Docker Compose.

## Features

- Add, view, and delete todos
- Frontend and backend services
- Fully containerized with Docker Compose

## Prerequisites

- Docker
- Docker Compose

## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/robbopp/docker-beginner-todolist.git
cd docker-beginner-todolist
```

2. Start the app:

```bash
docker compose build
docker compose up -d
```

3. Open the app in your browser.

4. Stop the app:

```bash
docker compose down
```
